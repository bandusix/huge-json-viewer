Всё, что нужно

Просмотрщик больших JSON, который работает с JSON как с данными, а не как с текстом

Все главные возможности профессионального инструмента для JSON, созданного для файлов гигабайтного масштаба и полностью бесплатного.

Гигабайты без разбиения на части

Отображает файл в память и строит потоковый индекс — открывает JSON 2–3 GB за секунды, не роняя ваш редактор.

Мгновенный поиск с RegEx

Ищите по ключам и значениям, обычным текстом или регулярным выражением, с живым счётчиком совпадений и переходом к совпадению, автоматически разворачивающим путь.

Наглядное дерево для чтения

Сворачиваемое дерево с подсветкой синтаксиса, номерами строк, направляющими отступов, цветами типов и счётчиками потомков — просматривайте от корня до листа.

Конвертация, извлечение и копирование

Выгружайте любой узел в готовый для таблиц CSV или в XML, извлекайте поддерево как JSON или копируйте любой ключ, значение или путь — работает с многогигабайтными файлами при ограниченном объёме памяти.

Объединение нескольких файлов

Открывайте сразу несколько JSON-файлов как одно общее дерево — каждый помечен именем файла — и ищите по ним всем.

Минимальный расход памяти

Индекс использует в RAM лишь ~1.5–2× от размера файла; сам файл остаётся высвобождаемым кэшем страниц ОС.

JSON Lines и NDJSON

Построчные и склеенные JSON-логи распознаются автоматически и оборачиваются в просматриваемый массив.

20 языков, поддержка RTL

Интерфейс поставляется на 20 языках, включая полную поддержку письма справа налево для арабского, урду и панджаби.

Приватно и офлайн

Ни сервера, ни загрузки, ни телеметрии. Ваш JSON никогда не покидает ваш компьютер.

Как это устроено

Почему он не захлёбывается на гигабайтах

1

Отображаем в память, а не загружаем

Файл отображается в память и подгружается ОС по мере необходимости — никогда не разбирается в прожорливые до RAM объекты.

2

Потоково строим компактный индекс

За один проход строится плоский структурный индекс байтовых смещений, поэтому структура известна без хранения данных.

3

Отрисовываем только то, что видно

Дерево полностью виртуализировано — материализуются только строки на экране, поэтому прокрутка миллионов строк остаётся плавной.

Вопросы и ответы

Открытие больших JSON-файлов — отвечаем на вопросы

Как открыть большой JSON-файл на Mac?

Скачайте BigJSON (бесплатно), затем перетащите ваш файл .json в окно или нажмите ⌘O. Он отображает файл в память и строит потоковый индекс, поэтому JSON 2–3 GB открывается за несколько секунд — без вылетов и без разбиения на части.

BigJSON действительно бесплатен?

Да — на 100% бесплатный и с открытым исходным кодом под лицензией MIT, включая коммерческое использование. Никаких платных тарифов, регистрации или пробного периода.

Насколько большой JSON-файл он может открыть?

До 4 GB на файл (и 4 GB суммарно при объединении нескольких файлов). Индекс использует в RAM примерно 1.5–2× от размера файла; машина с 16 GB комфортно справляется с файлами 2–3 GB.

Это хорошая бесплатная альтернатива Dadroit?

Да. Он охватывает те же ключевые возможности — сворачиваемое дерево, поиск по ключам и значениям с RegEx, экспорт JSON → CSV / XML и объединение нескольких файлов — бесплатно и с открытым исходным кодом, на macOS и Windows.

Может ли он конвертировать JSON в CSV или XML?

Да. Он выгружает JSON в CSV (массив объектов превращается в таблицу) или в XML и работает с многогигабайтными файлами при ограниченном объёме памяти.

Мои данные куда-нибудь загружаются?

Нет. Он работает на 100% офлайн — ни сервера, ни загрузки, ни телеметрии. Ваш JSON никогда не покидает ваш компьютер.

Работает ли он на Windows?

Да. BigJSON работает нативно на Windows 10 и 11 (64-бит), с тем же деревом просмотра, поиском по ключам и значениям и экспортом JSON → CSV / XML, что и в сборке для Mac. Скачайте установщик для Windows со страницы релизов и, если появится SmartScreen, выберите Подробнее → Выполнить в любом случае.
