Semua yang Anda butuhkan

Penampil JSON besar yang memperlakukan JSON sebagai data — bukan teks

Semua hal esensial dari alat JSON profesional, dibangun untuk file berskala gigabyte dan sepenuhnya gratis.

Gigabyte tanpa perlu dipecah

Melakukan memory-map pada file dan membangun indeks streaming — membuka JSON 2–3 GB dalam hitungan detik tanpa membuat editor Anda crash.

Pencarian instan dengan RegEx

Cari kunci dan nilai, teks biasa atau ekspresi reguler, dengan jumlah kecocokan langsung dan lompat-ke-kecocokan yang membentangkan jalur secara otomatis.

Penampil pohon yang mudah dibaca

Pohon yang dapat diciutkan dengan penyorotan sintaks, nomor baris, panduan indentasi, warna tipe, dan jumlah anak — jelajahi dari akar hingga daun.

Konversi, ekstrak & salin

Streaming node apa pun ke CSV yang siap untuk spreadsheet atau ke XML, tarik keluar subpohon sebagai JSON, atau salin kunci, nilai, atau jalur apa pun — pada file multi-gigabyte dengan penggunaan memori yang terbatas.

Gabungkan beberapa file

Buka beberapa file JSON sekaligus sebagai satu pohon gabungan — masing-masing diberi label nama file — dan cari di seluruhnya.

Jejak memori yang sangat kecil

Indeks hanya menggunakan ~1.5–2× dari ukuran file di RAM; file itu sendiri tetap sebagai cache halaman OS yang dapat diklaim kembali.

JSON Lines & NDJSON

Log JSON yang dibatasi baris dan digabungkan terdeteksi secara otomatis dan dibungkus menjadi array yang dapat dijelajahi.

20 bahasa, mendukung RTL

Antarmuka tersedia dalam 20 lokal, termasuk dukungan penuh dari kanan ke kiri untuk bahasa Arab, Urdu, dan Punjabi.

Privat & offline

Tanpa server, tanpa unggahan, tanpa telemetri. JSON Anda tidak pernah meninggalkan komputer Anda.

Gratis vs. berbayar

BigJSON vs. Dadroit

Perbandingan langsung, fitur demi fitur. Tingkat gratis Dadroit membatasi file hingga 50 MB dan batas yang lebih besar, penggabungan, serta penggunaan komersialnya berbayar — BigJSON memberikan semuanya secara cuma-cuma.

Di balik layar

Mengapa ia tidak tersendat pada gigabyte

1

Memory-map, jangan dimuat

File di-memory-map dan dipaginasi sesuai permintaan oleh OS — tidak pernah diurai menjadi objek yang boros RAM.

2

Streaming indeks yang ringkas

Satu kali proses membangun indeks struktural datar dari offset byte, sehingga struktur diketahui tanpa harus menyimpan data.

3

Render hanya yang Anda lihat

Pohon sepenuhnya divirtualisasi — hanya baris di layar yang dimaterialisasi, sehingga menggulir jutaan baris tetap mulus.

FAQ

Membuka file JSON besar, terjawab

Bagaimana cara membuka file JSON besar di Mac?

Unduh BigJSON (gratis), lalu seret file .json Anda ke jendela atau tekan ⌘O. Ia melakukan memory-map pada file dan membangun indeks streaming, sehingga JSON 2–3 GB terbuka dalam beberapa detik — tanpa crash, tanpa perlu dipecah.

Apakah BigJSON benar-benar gratis?

Ya — 100% gratis dan open source di bawah lisensi MIT, termasuk penggunaan komersial. Tanpa tingkatan berbayar, tanpa pendaftaran, tanpa uji coba.

Seberapa besar file JSON yang bisa dibuka?

Hingga 4 GB per file (dan 4 GB gabungan untuk penggabungan multi-file). Indeks menggunakan sekitar 1.5–2× dari ukuran file di RAM; mesin dengan 16 GB menangani file 2–3 GB dengan nyaman.

Apakah ini alternatif Dadroit gratis yang bagus?

Ya. Ia mencakup hal-hal esensial yang sama — penampil pohon yang dapat diciutkan, pencarian RegEx kunci dan nilai, ekspor JSON → CSV / XML, dan penggabungan beberapa file — secara gratis dan open source, di macOS dan Windows.

Bisakah ia mengonversi JSON ke CSV atau XML?

Ya. Ia melakukan streaming JSON ke CSV (array objek menjadi spreadsheet) atau ke XML, dan berfungsi pada file multi-gigabyte dengan penggunaan memori yang terbatas.

Apakah data saya diunggah ke suatu tempat?

Tidak. Ia 100% offline — tanpa server, tanpa unggahan, tanpa telemetri. JSON Anda tidak pernah meninggalkan komputer Anda.

Apakah ini berfungsi di Windows?

Ya. BigJSON berjalan secara native di Windows 10 dan 11 (64-bit), dengan penampil pohon, pencarian kunci & nilai, dan ekspor JSON → CSV / XML yang sama seperti build Mac. Unduh installer Windows dari halaman rilis dan, jika SmartScreen muncul, pilih More info → Run anyway.
