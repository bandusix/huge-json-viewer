Todo lo que necesitas

Un visor de JSON grande que trata el JSON como datos, no como texto

Todo lo esencial de una herramienta JSON profesional, creada para archivos a escala de gigabytes y totalmente gratis.

Gigabytes sin dividir

Mapea el archivo en memoria y construye un índice en streaming — abre JSON de 2–3 GB en segundos sin que tu editor se bloquee.

Búsqueda instantánea con RegEx

Busca en claves y valores, en texto plano o expresión regular, con recuento de coincidencias en vivo y salto a la coincidencia que expande la ruta automáticamente.

Visor de árbol legible para humanos

Árbol plegable y con resaltado de sintaxis con números de línea, guías de sangría, colores por tipo y recuento de hijos — explora desde la raíz hasta la hoja.

Convierte, extrae y copia

Envía en streaming cualquier nodo a un CSV listo para hoja de cálculo o a XML, extrae un subárbol como JSON, o copia cualquier clave, valor o ruta — en archivos de varios gigabytes con memoria acotada.

Une varios archivos

Abre varios archivos JSON a la vez como un único árbol combinado — cada uno etiquetado por nombre de archivo — y busca en todos ellos.

Huella de memoria mínima

El índice usa solo ~1.5–2× el tamaño del archivo en RAM; el archivo en sí permanece como caché de páginas del SO recuperable.

JSON Lines y NDJSON

Los registros JSON delimitados por líneas y concatenados se detectan automáticamente y se envuelven en un array navegable.

20 idiomas, compatible con RTL

La interfaz se ofrece en 20 idiomas, incluida la compatibilidad total de derecha a izquierda para árabe, urdu y punyabí.

Privado y sin conexión

Tu JSON nunca sale de tu ordenador: sin subir archivos, sin nube, nunca. Lo único que se envía son estadísticas de uso anónimas y opcionales que puedes desactivar.

Bajo el capó

Por qué no se atasca con los gigabytes

1

Mapea en memoria, no cargues

El archivo se mapea en memoria y el SO lo pagina bajo demanda — nunca se analiza en objetos que consumen RAM.

2

Genera un índice compacto en streaming

Una sola pasada construye un índice estructural plano de desplazamientos de bytes, así que la estructura se conoce sin retener los datos.

3

Renderiza solo lo que ves

El árbol está totalmente virtualizado — solo se materializan las filas en pantalla, así que desplazarse por millones de filas se mantiene fluido.

Preguntas frecuentes

Cómo abrir archivos JSON grandes, respondido

¿Cómo abro un archivo JSON grande en Mac?

Descarga BigJSON (gratis) y luego arrastra tu archivo .json a la ventana o pulsa ⌘O. Mapea el archivo en memoria y construye un índice en streaming, así que un JSON de 2–3 GB se abre en unos segundos — sin bloqueos, sin dividir.

¿BigJSON es realmente gratis?

Sí — 100% gratis y de código abierto bajo la licencia MIT, incluido el uso comercial. Sin planes de pago, sin registro, sin prueba.

¿Qué tamaño de archivo JSON puede abrir?

Hasta 4 GB por archivo (y 4 GB combinados para una unión de varios archivos). El índice usa aproximadamente 1.5–2× el tamaño del archivo en RAM; una máquina de 16 GB maneja archivos de 2–3 GB con holgura.

¿Es una buena alternativa gratuita a Dadroit?

Sí. Cubre lo mismo esencial — un visor de árbol plegable, búsqueda RegEx de claves y valores, exportación de JSON → CSV / XML y unión de varios archivos — de forma gratuita y de código abierto, en macOS y Windows.

¿Puede convertir JSON a CSV o XML?

Sí. Envía JSON en streaming a CSV (un array de objetos se convierte en una hoja de cálculo) o a XML, y funciona con archivos de varios gigabytes con memoria acotada.

¿Se sube mis datos a algún sitio?

Tu JSON nunca lo hace. Los archivos —nombres y contenidos— permanecen 100% en tu ordenador: sin subidas, sin servidor, nunca. BigJSON solo envía estadísticas de uso anónimas y opcionales que puedes desactivar (qué funciones se usan, nunca datos de archivos), y con un clic las desactivas.

¿Funciona en Windows?

Sí. BigJSON se ejecuta de forma nativa en Windows 10 y 11 (64 bits), con el mismo visor de árbol, búsqueda de claves y valores y exportación de JSON → CSV / XML que la versión para Mac. Descarga el instalador de Windows desde la página de versiones y, si aparece SmartScreen, elige Más información → Ejecutar de todos modos.
