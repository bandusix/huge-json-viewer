底层原理

为什么它面对数 GB 也不卡顿

1

内存映射，而非加载

文件通过内存映射，由操作系统按需分页读入——绝不解析成占用大量内存的对象。

2

流式构建紧凑索引

一次扫描即可构建字节偏移的扁平结构索引，因此无需保留数据即可掌握其结构。

3

只渲染你看到的部分

树形视图完全虚拟化——只有屏幕上的行会被实例化，因此滚动数百万行依然流畅。

常见问题

关于打开超大 JSON 文件的解答

如何在 Mac 上打开一个超大 JSON 文件？

下载 BigJSON（免费），然后把你的 .json 文件拖到窗口上，或按 ⌘O。它会对文件做内存映射并构建流式索引，因此 2–3 GB 的 JSON 只需几秒即可打开——不崩溃，也无需拆分。

BigJSON 真的免费吗？

是的——基于 MIT 许可 100% 免费开源，包括商业使用。没有付费版本，无需注册，也没有试用期。

它能打开多大的 JSON 文件？

单个文件最大 4 GB（多文件合并后合计也是 4 GB）。索引在内存中大约占用文件大小的 1.5–2×；16 GB 内存的电脑可轻松处理 2–3 GB 的文件。

它是 Dadroit 的理想免费替代方案吗？

是的。它涵盖了同样的核心功能——可折叠的树形查看器、键与值的 RegEx 搜索、JSON → CSV / XML 导出，以及多文件合并——免费且开源，同时支持 macOS 和 Windows。

它能把 JSON 转换成 CSV 或 XML 吗？

可以。它能将 JSON 流式导出为 CSV（对象数组会变成电子表格）或 XML，并在内存占用可控的前提下处理数 GB 级文件。

我的数据会被上传到什么地方吗？

你的 JSON 绝不会。文件——包括名称和内容——100% 保留在你的电脑上：不上传、无服务器。BigJSON 仅发送匿名、可随时关闭的使用统计（哪些功能被使用——绝不涉及任何文件数据），一键即可关闭。

它能在 Windows 上运行吗？

可以。BigJSON 在 Windows 10 和 11（64 位）上原生运行，提供与 Mac 版相同的树形查看器、键与值搜索，以及 JSON → CSV / XML 导出。请从 releases 页面下载 Windows 安装程序；若出现 SmartScreen，请选择 更多信息 → 仍要运行。
