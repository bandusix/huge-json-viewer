底層原理

為什麼它面對數 GB 也不會卡頓

1

記憶體映射，而非載入

檔案透過記憶體映射，由作業系統按需分頁讀入——絕不解析成佔用大量記憶體的物件。

2

串流建立精簡索引

一次掃描即可建立位元組偏移的扁平結構索引，因此無需保留資料即可掌握其結構。

3

只繪製你看到的部分

樹狀檢視完全虛擬化——只有畫面上的列會被具現化，因此捲動數百萬列依然流暢。

常見問題

關於開啟超大 JSON 檔案的解答

如何在 Mac 上開啟一個超大 JSON 檔案？

下載 BigJSON（免費），然後把你的 .json 檔案拖到視窗上，或按 ⌘O。它會對檔案做記憶體映射並建立串流索引，因此 2–3 GB 的 JSON 只需幾秒即可開啟——不會當機，也無需分割。

BigJSON 真的免費嗎？

是的——採用 MIT 授權 100% 免費開放原始碼，包括商業使用。沒有付費版本，無需註冊，也沒有試用期。

它能開啟多大的 JSON 檔案？

單一檔案最大 4 GB（多檔案合併後合計也是 4 GB）。索引在記憶體中大約佔用檔案大小的 1.5–2×；16 GB 記憶體的電腦可輕鬆處理 2–3 GB 的檔案。

它是 Dadroit 的理想免費替代方案嗎？

是的。它涵蓋了同樣的核心功能——可折疊的樹狀檢視器、鍵與值的 RegEx 搜尋、JSON → CSV / XML 匯出，以及多檔案合併——免費且開放原始碼，同時支援 macOS 與 Windows。

它能把 JSON 轉換成 CSV 或 XML 嗎？

可以。它能將 JSON 串流匯出為 CSV（物件陣列會變成試算表）或 XML，並在記憶體用量可控的前提下處理數 GB 級檔案。

我的資料會被上傳到任何地方嗎？

不會。它 100% 離線運作——無伺服器、無上傳、無遙測。你的 JSON 永遠不會離開你的電腦。

它能在 Windows 上運作嗎？

可以。BigJSON 在 Windows 10 和 11（64 位元）上原生執行，具備與 Mac 版相同的樹狀檢視器、鍵與值搜尋，以及 JSON → CSV / XML 匯出。請從發布頁面下載 Windows 安裝程式，若出現 SmartScreen，請選擇「更多資訊 → 仍要執行」。
