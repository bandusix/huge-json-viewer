Tudo o que você precisa

Um visualizador de JSON grande que trata JSON como dados — não como texto

Todos os fundamentos de uma ferramenta JSON profissional, feita para arquivos na escala de gigabytes e totalmente gratuita.

Gigabytes sem dividir

Mapeia o arquivo na memória e constrói um índice em streaming — abre JSON de 2–3 GB em segundos sem travar seu editor.

Busca instantânea com RegEx

Pesquise chaves e valores, em texto simples ou expressão regular, com contagem de resultados em tempo real e salto para o resultado que expande o caminho automaticamente.

Visualizador em árvore legível

Árvore recolhível com destaque de sintaxe, números de linha, guias de indentação, cores por tipo e contagem de filhos — navegue da raiz até a folha.

Converta, extraia e copie

Exporte qualquer nó em streaming para um CSV pronto para planilhas ou para XML, extraia uma subárvore como JSON, ou copie qualquer chave, valor ou caminho — em arquivos de vários gigabytes com memória limitada.

Una múltiplos arquivos

Abra vários arquivos JSON de uma vez como uma única árvore combinada — cada um identificado pelo nome do arquivo — e pesquise em todos eles.

Uso mínimo de memória

O índice usa apenas ~1,5–2× o tamanho do arquivo em RAM; o próprio arquivo permanece como cache de páginas do sistema, recuperável a qualquer momento.

JSON Lines e NDJSON

Logs JSON delimitados por linha e concatenados são detectados automaticamente e agrupados em um array navegável.

20 idiomas, com suporte a RTL

A interface está disponível em 20 idiomas, incluindo suporte completo da direita para a esquerda para árabe, urdu e panjabi.

Privado e offline

Seu JSON nunca sai do seu computador — sem upload de arquivos, sem nuvem, nunca. A única coisa enviada são estatísticas de uso anônimas e desativáveis.

Nos bastidores

Por que ele não engasga com gigabytes

1

Mapeie na memória, não carregue

O arquivo é mapeado na memória e paginado sob demanda pelo sistema operacional — nunca convertido em objetos que consomem RAM.

2

Gere um índice compacto em streaming

Uma única passagem constrói um índice estrutural plano de deslocamentos de bytes, então a estrutura é conhecida sem manter os dados na memória.

3

Renderize apenas o que você vê

A árvore é totalmente virtualizada — apenas as linhas na tela são materializadas, então rolar por milhões de linhas permanece fluido.

Perguntas frequentes

Como abrir arquivos JSON grandes, respondido

Como abro um arquivo JSON grande no Mac?

Baixe o BigJSON (gratuito), depois arraste seu arquivo .json para a janela ou pressione ⌘O. Ele mapeia o arquivo na memória e constrói um índice em streaming, então um JSON de 2–3 GB abre em poucos segundos — sem travar, sem dividir.

O BigJSON é realmente gratuito?

Sim — 100% gratuito e de código aberto sob a licença MIT, incluindo uso comercial. Sem planos pagos, sem cadastro, sem período de teste.

Qual o tamanho máximo de arquivo JSON que ele consegue abrir?

Até 4 GB por arquivo (e 4 GB no total para uma união de múltiplos arquivos). O índice usa aproximadamente 1,5–2× o tamanho do arquivo em RAM; uma máquina com 16 GB lida com arquivos de 2–3 GB tranquilamente.

É uma boa alternativa gratuita ao Dadroit?

Sim. Ele cobre os mesmos fundamentos — um visualizador em árvore recolhível, busca por chave e valor com RegEx, exportação de JSON → CSV / XML e união de múltiplos arquivos — de forma gratuita e de código aberto, no macOS e no Windows.

Ele consegue converter JSON para CSV ou XML?

Sim. Ele exporta JSON em streaming para CSV (um array de objetos vira uma planilha) ou para XML, e funciona em arquivos de vários gigabytes com memória limitada.

Meus dados são enviados para algum lugar?

Seu JSON nunca é. Os arquivos — nomes e conteúdos — permanecem 100% no seu computador: sem upload, sem servidor, nunca. O BigJSON envia apenas estatísticas de uso anônimas e desativáveis (quais recursos são usados — nunca qualquer dado de arquivo), e um clique desativa isso.

Ele funciona no Windows?

Sim. O BigJSON roda nativamente no Windows 10 e 11 (64 bits), com o mesmo visualizador em árvore, busca por chave e valor e exportação de JSON → CSV / XML da versão para Mac. Baixe o instalador do Windows na página de versões e, se o SmartScreen aparecer, escolha Mais informações → Executar assim mesmo.
