كل ما تحتاجه

عارض JSON كبير يتعامل مع JSON كبيانات — لا كنص

كل أساسيات أداة JSON احترافية، مصمّمة لملفات بحجم الغيغابايت ومجانية تمامًا.

غيغابايتات دون تقسيم

يربط الملف بالذاكرة ويبني فهرسًا متدفقًا — يفتح JSON بحجم 2–3 GB في ثوانٍ دون أن ينهار محرّرك.

بحث فوري بتعبيرات RegEx

ابحث في المفاتيح والقيم، بنص عادي أو تعبير نمطي، مع عدّاد مطابقات حيّ وانتقال إلى المطابقة يوسّع المسار تلقائيًا.

عارض شجري مقروء للبشر

شجرة قابلة للطي مع تمييز لوني للصياغة وأرقام الأسطر وأدلة المسافات البادئة وألوان الأنواع وأعداد العناصر الفرعية — تصفّح من الجذر إلى الورقة.

تحويل واستخراج ونسخ

صدّر أي عقدة كتدفّق إلى ملف CSV جاهز لجداول البيانات أو إلى XML، أو استخرج شجرة فرعية بصيغة JSON، أو انسخ أي مفتاح أو قيمة أو مسار — على ملفات بحجم عدة غيغابايت باستهلاك ذاكرة محدود.

دمج ملفات متعددة

افتح عدة ملفات JSON دفعة واحدة كشجرة موحّدة واحدة — كلٌّ موسوم باسم ملفه — وابحث عبرها جميعًا.

بصمة ذاكرة ضئيلة

يستخدم الفهرس نحو ~1.5–2× من حجم الملف في الذاكرة؛ ويبقى الملف نفسه في ذاكرة التخزين المؤقت للصفحات القابلة للاستعادة في نظام التشغيل.

JSON Lines و NDJSON

يُكتشف سجلّ JSON المفصول بأسطر والمتسلسل تلقائيًا ويُغلَّف في مصفوفة قابلة للتصفح.

20 لغة، مع دعم RTL

تتوفر الواجهة بـ 20 لغة، بما في ذلك دعم كامل للاتجاه من اليمين إلى اليسار للعربية والأردية والبنجابية.

خاص وبلا اتصال

لا يغادر ملف JSON جهاز الكمبيوتر الخاص بك أبدًا — بلا رفع للملفات، بلا سحابة، على الإطلاق. الشيء الوحيد الذي يُرسَل هو إحصاءات استخدام مجهولة الهوية يمكن إيقافها.

خلف الكواليس

لماذا لا يختنق أمام الغيغابايتات

1

اربط بالذاكرة، لا تُحمّل

يُربط الملف بالذاكرة ويُحمّله نظام التشغيل صفحة تلو الأخرى عند الطلب — دون تحليله أبدًا إلى كائنات نهمة للذاكرة.

2

تدفّق فهرس مضغوط

تبني تمريرة واحدة فهرسًا بنيويًا مسطّحًا لإزاحات البايتات، فتُعرف البنية دون الاحتفاظ بالبيانات.

3

اعرض ما تراه فقط

الشجرة مُحوّلة افتراضيًا بالكامل — تُنشأ الصفوف الظاهرة على الشاشة فقط، فيبقى تمرير ملايين الصفوف سلسًا.

الأسئلة الشائعة

فتح ملفات JSON الكبيرة، بإجابات وافية

كيف أفتح ملف JSON كبير على جهاز Mac؟

نزّل BigJSON (مجانًا)، ثم اسحب ملف .json الخاص بك إلى النافذة أو اضغط ⌘O. يربط الملف بالذاكرة ويبني فهرسًا متدفقًا، فيُفتح JSON بحجم 2–3 GB في بضع ثوانٍ — بلا انهيار، بلا تقسيم.

هل BigJSON مجاني حقًا؟

نعم — مجاني ومفتوح المصدر 100% بموجب ترخيص MIT، بما في ذلك الاستخدام التجاري. بلا طبقات مدفوعة، بلا تسجيل، بلا فترة تجريبية.

ما أقصى حجم لملف JSON يمكنه فتحه؟

حتى 4 GB لكل ملف (و 4 GB مجتمعة لدمج عدة ملفات). يستخدم الفهرس نحو 1.5–2× من حجم الملف في الذاكرة؛ ويتعامل جهاز بذاكرة 16 GB مع ملفات 2–3 GB بسهولة.

هل هو بديل مجاني جيد لـ Dadroit؟

نعم. فهو يغطي الأساسيات ذاتها — عارض شجري قابل للطي، وبحث في المفاتيح والقيم بتعبيرات RegEx، وتصدير JSON ← CSV / XML، ودمج ملفات متعددة — مجانًا ومفتوح المصدر، على macOS و Windows.

هل يمكنه تحويل JSON إلى CSV أو XML؟

نعم. يصدّر JSON إلى CSV (تصبح مصفوفة الكائنات جدول بيانات) أو إلى XML، ويعمل على ملفات بحجم عدة غيغابايت باستهلاك ذاكرة محدود.

هل تُرفع بياناتي إلى أي مكان؟

لا يُرفع ملف JSON الخاص بك أبدًا. تبقى الملفات — أسماؤها ومحتوياتها — على جهاز الكمبيوتر الخاص بك بنسبة 100%: بلا رفع، بلا خادم، على الإطلاق. لا يرسل BigJSON سوى إحصاءات استخدام مجهولة الهوية يمكن إيقافها (أي المزايا التي تُستخدَم — دون أي بيانات من الملفات إطلاقًا)، ونقرة واحدة توقفها.

هل يعمل على Windows؟

نعم. يعمل BigJSON أصليًا على Windows 10 و 11 (64 بت)، بنفس العارض الشجري، والبحث في المفاتيح والقيم، وتصدير JSON ← CSV / XML الموجودة في نسخة Mac. نزّل مُثبّت Windows من صفحة الإصدارات، وإذا ظهر SmartScreen، اختر مزيد من المعلومات ← التشغيل على أي حال.
